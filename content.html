To include a hyperlink in an HTML document, you use the <a> tag. For example:

<a href="https://www.imperial.ac.uk">Imperial College London</a>

This will render as "Imperial College London". The text within the tags is the link text that will be displayed to the user. The href attribute specifies the URL of the page the link goes to.

Relative Links

In the example above, we used an absolute URL (starting with https://). You can also use a relative URL, which is relative to the current page. For example:

<a href="about.html">About Us</a>
